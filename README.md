# 🛒 FreshMart – Grocery Shopping Web Application

FreshMart is a modern grocery shopping web application built with **React.js** and **Redux Toolkit**. It provides a seamless shopping experience with secure authentication, product browsing, cart management, and checkout functionality.

## ✨ Features

* 🔐 User Registration & Login
* 🛍️ Browse Grocery Products
* 📦 Product Details Page
* 🛒 Add to Cart / Remove from Cart
* 💳 Checkout Page
* 🔒 Protected Routes
* ⚡ Redux Toolkit State Management
* 📱 Responsive Design
* ✅ Form Validation
* 🛡️ Basic Security Utilities

## 🛠️ Tech Stack

* React.js
* Redux Toolkit
* React Router DOM
* JavaScript (ES6+)
* SCSS
* HTML5
* CSS3

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/anshulchoudhary25/freshmart-grocery-app.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```

The application will run at:

```text
http://localhost:3000
```
